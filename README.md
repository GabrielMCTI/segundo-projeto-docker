# Docker Swarm Cluster com Vagrant

Projeto de cluster Docker Swarm utilizando Vagrant e VirtualBox para criação de máquinas virtuais.

## 🚀 Tecnologias utilizadas

- Vagrant
- VirtualBox
- Docker
- Docker Swarm
- Ubuntu 22.04

## 🧱 Arquitetura

- master (manager)
- node01 (worker)
- node02 (worker)

## 🌐 Rede

- master: 10.10.10.100
- node01: 10.10.10.101
- node02: 10.10.10.102

## ⚙️ Como executar

```bash
vagrant up
